#这个是自定义右键菜单的插件，欢迎使用，该插件目前还不完善，我会继续更新
*1：使用该插件之前，你应该首先定义一个json变量，类似如下这种
var json={"add_h1_nav":"添加下级菜单栏","delete_nav":"删除","cut":"剪切","copy":"粘贴","change_nav":"编辑"};
*2：注意，不能改变上面json文件中的英语也就是key值，汉子随便更改，如果你只是需要其中的一些功能，只需要删除json
变量中的键值对就可以了
*3：new charts().more_menu("gps",json,"nav");这就是在该插件被调用之后的使用方法，
gps代表右键触发的区域（gps应该为id），nav为右键菜单内容显示的区域（nav同样是id）
